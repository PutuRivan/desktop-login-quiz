Pengenalan Aplikasi

Aplikasi ini digunakan untuk monitoring perangkat kantor. Anda wajib memahami cara penggunaan sistem sebelum melanjutkan login sebagai admin.

Aturan Penggunaan

  • Dilarang menutup aplikasi sebelum proses selesai.
  • Dilarang mengubah konfigurasi sistem.
  • Semua aktivitas dimonitor oleh sistem keamanan.
